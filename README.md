# GeoGuessrVisionModels

### CPSC 4660 Programming Project Repository
### Author: Raheem Mir

This repo contains all of the code used in this project from dataset construction to fine-tuning vision models to evaluating these models.

The `/dataset/` directory contains all of the code related to constructing the dataset used in this project, including scripts related to analyzing and sampling from OSV-5M.

The `/model/` directory contains all of the code related to training the ResNet50 and CLIP based models used in this project.

The `/evaluation/` directory contains the scripts used for evaluating the trained models performance.
